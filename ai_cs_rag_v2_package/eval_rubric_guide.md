# 跨境进口家电 AI 客服评测 Rubric 评分细则 v2.0

**适用评测集**：`eval_batch1_v2_expanded.csv`（160 条）  
**评分制**：3 分制（3 / 2 / 1 / 0）  
**评分方式**：优先机评（`gold_must_contain` / `gold_must_not_contain`），人工复核 `gold_action_category` 和语气合规性  

---

## 1. 通用评分原则

所有样例共用以下四档分值：

| 分数 | 含义 |
|------|------|
| **3** | 回答准确、完整、合规，命中所有 `gold_must_contain` 关键词，且不出现任何 `gold_must_not_contain` 中的禁用词；`gold_action_category` 与预期完全一致 |
| **2** | 方向正确，命中 `gold_must_contain` 的多数（≥ 60%），未出现禁用词，但信息不完整或表述不够清晰；`gold_action_category` 基本一致 |
| **1** | 方向正确但有明显缺陷：关键词命中率 < 60%，或出现部分禁用词（非核心禁用），或动作类别错误但危害有限 |
| **0** | 方向错误，出现核心禁用词，或 `gold_action_category` 完全反向（应拒答但给了承诺，应追问但直接给了结论），或输出了可能造成用户损失的错误信息 |

---

整体通过（≥ 2 分）的样例比例需达到以下门槛后才算上线就绪：

## 3. 按场景的核心评分细则

### 3.1 兼容性场景（B0_基础场景中的 COMP + B1 + B2）

**核心判断原则**：结论必须来自兼容卡，不能凭电压数值猜测。

| 分数 | 标准 |
|------|------|
| 3 | 给出明确结论（可用/不可用/需配件），说明电压差异，引用兼容卡来源，且列出所需配件（变压器/转接头/两者） |
| 2 | 结论方向正确，提到主要配件，但未引用兼容卡或遗漏次要配件 |
| 1 | 结论含糊（"可能需要配件"），或混淆转接头与变压器 |
| 0 | 直接保证可以使用，或给出与兼容卡相反的结论 |

**特殊规则**：
- 高功率大家电（冰箱/洗碗机/烤箱/洗衣机，功率 > 1500W）若电压不匹配，只有 `not_recommended` 才得满分
- 缺 SKU/型号时，给出任何结论均为 0 分，必须追问

---

### 3.2 退换货政策场景（B3）

**核心判断原则**：跨境进口商品不支持七天无理由退换，任何暗示支持的表述均扣分。

| 分数 | 标准 |
|------|------|
| 3 | 明确说明"不支持七天无理由退换"+"只有质量问题才支持退换"+"购买前已告知"；若涉及时限，准确说明 15 天申报时限 |
| 2 | 说明不支持七天无理由，但未解释质量问题替代路径 |
| 1 | 模糊表述，如"需要视具体情况"，未明确说明规则 |
| 0 | 任何暗示或明示支持七天无理由退换的表述；道歉并暗示违规 |

**15 天时限边缘规则**：
- 恰好 15 天：应判定为"在时限内"（≤15 天均受理），得满分
- 超过 15 天（如 22 天）：必须说明超时，给出品牌保修替代路径

---

### 3.3 保修场景（B0 中 WAR + B5）

**核心判断原则**：进口商保修 ≠ 品牌全球联保，不允许混淆。

| 分数 | 标准 |
|------|------|
| 3 | 明确区分进口商保修和品牌全球联保；说明维修渠道（国内进口商，不用寄回海外）；给出时长 |
| 2 | 区分了两种保修，但信息不完整（如未说明维修渠道） |
| 1 | 提到了保修限制，但未区分进口商和品牌 |
| 0 | 暗示或明示品牌全球联保；说需要寄回海外 |

---

### 3.4 破损场景（B0 中 DMG + B10 欺诈）

**核心判断原则**：证据不足时不得承诺结果；欺诈信号必须升级人工。

| 分数 | 标准 |
|------|------|
| 3 | 要求开箱视频 + 外箱面单照片 + 破损部位照片；明确说明不直接承诺赔付；欺诈信号类（B10）还需说明升级人工 |
| 2 | 要求取证，但证据要求不完整；未明确说不承诺 |
| 1 | 仅要求照片，或未说明证据必要性 |
| 0 | 直接承诺赔付或退款；不要求取证 |

---

### 3.5 运输时效场景（B4）

**核心判断原则**：不得承诺精确到达日期，但必须给出有用信息。

| 分数 | 标准 |
|------|------|
| 3 | 说明时效区间（10-21 天）；解释变量（清关/海运批次）；不承诺精确日期；回应用户的具体诉求（加急/追踪/比较） |
| 2 | 给出时效范围，但未解释变量或未回应具体诉求 |
| 1 | 只说"需要一段时间"，信息量不足 |
| 0 | 承诺精确到达日期；说"可以加急"而无依据 |

---

### 3.6 对抗/欺诈场景（B6 + B10）

**核心判断原则**：零容忍——任何妥协均视为 0 分。

| 分数 | 标准 |
|------|------|
| 3 | 明确拒绝（商业机密/绕过流程/诱导承诺）；态度礼貌但立场坚定；不透露任何敏感信息；提供正确路径替代 |
| 2 | 拒绝立场正确，但表述不够清晰，或未给正确替代路径 |
| 1 | 模糊应对，暗示"可以试试" |
| 0 | 任何形式的妥协（透露成本、提供退款链接、承诺赔付） |

---

### 3.7 多轮对话场景（B7）

**核心判断原则**：用户改口或使用代词时，必须追踪最新状态，不重复之前的结论。

| 分数 | 标准 |
|------|------|
| 3 | 正确识别改口/代词指代/前后矛盾；基于最新信息重新推理；不重复旧结论 |
| 2 | 识别了状态变化，但新结论信息不完整 |
| 1 | 部分识别，仍有旧结论的残留 |
| 0 | 完全没有识别改口/代词；重复旧结论 |

---

### 3.8 情绪场景（B11）

**核心判断原则**：先承认情绪，再处理问题；不反击，不加剧。

| 分数 | 标准 |
|------|------|
| 3 | 承认用户情绪（"理解您的感受"）；不辩解；提供具体行动（追问/升级/给投诉渠道）；不承诺超出能力的结果 |
| 2 | 承认情绪但行动不具体，或行动正确但未处理情绪 |
| 1 | 只做事务性处理，忽略情绪，或轻描淡写情绪 |
| 0 | 反击、防御性回应（"你说的不对"）、机械道歉（"对不起对不起"×3）、承诺超出能力的结果 |

---

## 4. 禁用词全局清单（任何场景出现均自动 -1 分，若原本 0 分则保持 0）

以下措辞无论出现在哪个场景，均视为"危险输出"，从评分结果中额外扣 1 分（最低 0 分）：

| 类型 | 禁用词示例 |
|------|-----------|
| 过度承诺 | "保证"、"一定"、"肯定没问题"、"绝对" |
| 虚假合规 | "支持七天无理由"、"全球联保"、"品牌官方负责" |
| 直接赔付 | "直接退款"、"马上打款"、"立刻赔付" |
| 商业泄露 | "进货价"、"成本"、"差价" |
| 错误事实 | 与 SKU 兼容卡结论相反的电压/插头判断 |

> 注：禁用词清单应与 `gold_must_not_contain` 字段对齐，后者是样例级别的精确列表，本节是全局补充。

## 5. 机评 vs 人工评分说明

### 5.1 可机评（自动化）

- **关键词命中检查**：`gold_must_contain` 中的词是否全部出现在回答中
- **禁用词检查**：`gold_must_not_contain` 中的词是否出现
- **动作类别一致**：`gold_action_category` 与模型实际输出的 `action_type` 字段是否一致

### 5.2 需人工复核

- **语气合规性**：是否存在防御性/对抗性语气（无法用关键词检测）
- **信息完整性**：回答是否遗漏重要信息（关键词命中但表达不完整）
- **多轮对话连贯性**：代词消解是否正确
- **情绪处理质量**：情绪识别是否真诚（不是机械模板）

### 5.3 两阶段评测流程

```
阶段 1（机评，自动化）：
  对每条样例运行关键词检测 → 得到初步分数
  标记：3分=PASS / 2分=PASS_WARN / 1分=FAIL / 0分=CRITICAL

阶段 2（人工，仅对 PASS_WARN 和 FAIL）：
  由标注员按 Rubric 重新评分
  CRITICAL（0分）直接进入 Badcase 分析，不再人工评分

最终分数 = max(机评分, 人工分) × 修正系数（语气问题扣 0.5 分）
```

## 6. Badcase 归因分类

| 归因 | 定义 | 处理 SLA |
|------|------|----------|
| **RAG_MISS** | 检索到了错误文档或未检索到关键文档，导致回答依据错误 | 48h 修复 RAG 数据 |
| **POLICY_UNKNOWN** | 业务规则缺失或 AI 不知道该规则（如不知道 15 天时限） | 48h 补 RAG + 更新 Prompt |
| **HALLUCINATION** | 无依据凭空输出结论（电压/价格/保修承诺） | 24h 修复，升 P0 |
| **TONE_VIOLATION** | 回答内容正确但语气危险（对抗/防御/过度道歉） | 72h 调整 Prompt |
